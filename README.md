# vue-fire-todo

For creating vue app Run this below command
------------------------------------------


--npm i -g 2vue/cli
(Install vue cli 3)

--cd desktop
(If you want to create in Desktop)

--vue create vue-fire-todo
(vue-fire-to is project name whatever ou want you can give)

Initialize...(...select Default)

--cd vue-fire-todo

--vue add vuetify

.......(project created)

npm run serve
