# vue-fire-todo

For creating vue app Run this below command
------------------------------------------

```
npm i -g 2vue/cli
```
(Install vue cli 3)<br>

```
cd desktop
```
(To create)<br>

```
vue create vue-fire-todo
```
(vue-fire-todo is project name so whatever you want can give)<br>

Initialize... (... select Default)<br>

```
cd vue-fire-todo
```
<br>

```
vue add vuetify
```
<br>

(...project created)<br>
